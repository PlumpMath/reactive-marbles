# reactive-marbles

Testing RxJava with executable marble diagrams

- RxJava: https://github.com/Netflix/RxJava
- Marble diagrams: http://netflix.github.com/RxJava/javadoc/rx/Observable.html

## Usage
```
lein test reactive-marbles.rxjava-test
```
